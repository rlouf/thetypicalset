# Colophon

These notes are written in [Obsidian](https://obsidian.md/) and published with [Quartz](https://quartz.jzhao.xyz/). They're infused with a healthy dose of nostalgia for [the](https://www.spacejam.com/1996/) old [web](http://info.cern.ch/hypertext/WWW/TheProject.html); the design is essentially the same as this [motherfucking website](https://motherfuckingwebsite.com/), with a few lines of CSS to improve readability. I was heavily inspired by [Andy Matuschak's notes](https://notes.andymatuschak.org/About_these_notes) and [Neil Mather's digital garden](https://commonplace.doubleloop.net/).

The typical set is forever a work in progress. There used to be a list of things TODO in this section, but AI agents have allowed me to go through that list faster than new ideas have come up. What an amazing time to be alive!
