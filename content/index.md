# I write software...

Before diving into Large Language Models I was particularly interested in Bayesian Statistics and generative modeling, MCMC sampling and symbolic programming. The interests translated into the open source libraries I maintained:

- [Outlines](https://github.com/dottxt-ai/outlines) is a Python library to guide the generation of Large Language models.
- [[aesara|Aesara]] is a Python library for defining, rewriting and efficiently evaluating mathematical operations on arrays; A "tensor" library.
- [[aeppl|AePPL]] is a collection of tools for A[e]PPL in Aesara.
- [[blackjax|Blackjax]] is a sampling library built with JAX, focused on ease of use, speed and modularity.
- [[aemcmc|AeMCMC]] is a sampling library built with Aesara which builds symbolic samplers from model graphs.
- [Aehmc](https://github.com/aesara-devs/aehmc) provides implementations of samplers in the HMC family (including NUTS) with *Aesara*.
